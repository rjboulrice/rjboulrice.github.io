CrowdStrike BSOD Fix

This guide addresses a Blue Screen of Death (BSOD) issue caused by a problematic CrowdStrike Falcon Sensor update on a Windows 10 VM. It provides a step-by-step solution to boot the VM into Safe Mode via the Windows Recovery Environment (WinRE) and remove the faulty driver to resolve the error and restore normal operation.

KnowBe4 North Korean Insider Threat

This case study covers a KnowBe4 incident where a new hire, posing as a North Korean agent, tried to deploy malware. The threat was swiftly contained without compromising data. Key lessons include strengthening hiring processes, user training, and using tools like MFA, UEBA, and DLP for better threat prevention.
